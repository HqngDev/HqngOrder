# HqngOrder

A complete rewrite of the SBOrder plugin by BM Studio.
Decompiled from the original and reimplemented in Kotlin to provide improved functionality, performance, and maintainability.

Built with TriumphGUI API and FoliaLib for enhanced UI capabilities and optimized multithreading. All core features from SBOrder are preserved, with significant improvements in customization options ( like gui & config ) and overall system performance.
